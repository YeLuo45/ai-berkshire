# 2026-08 中报专题跟踪框架

**报告日期：2026-07-10(周五) | 中报季时间：2026-08-中旬至 2026-09-中旬 | 涉及公司：13+1 投资光谱中 9 家中报节点**

> 本文是为 2026-08 中报季设计的**专用跟踪框架**。在 7-9 月中报披露密集期,提供逐家公司的关键观察点 + 加减分判定标准。

---

## 0. 框架目标

**中报季是验证评级假设的关键窗口**。13+1 光谱中 9 家有 8 月中报节点,通过本框架:
1. **预先设定**每家公司的中报观察点(避免临时手忙脚乱)
2. **统一标准**(升/降级的具体条件)
3. **同步跟踪**(一次性对比多家公司)
4. **快速反应**(中报后 1-2 天内出更新报告)

## 2. 逐家公司中报观察清单

### 2.1 茅台 (600519.SH) — 信仰底仓

**核心问题**:2025 双降是周期性还是结构性?2026 Q1 恢复是否延续?

| 观察点 | 加分 | 减分 |
|--------|------|------|
| **营收同比** | +5%+(连续 2 季) | <+3% 或负增长 |
| **净利率** | 维持 47%+ | <45% |
| **批价(飞天)** | 回升至 ¥1,800+ | 跌至 ¥1,500 以下 |
| **出厂价** | 再次上调(继 2026-03 8.6% 之后) | **首次下调(25 年首例 = 极强减分)** |
| **渠道库存** | 持续消化 | 库存累积 |
| **管理层** | 稳定 | 重大动荡 |

**升级/降级规则**:
- 营收 +5%+ + 批价回升 → 维持 ★★★★,可加仓至 12-15%
- 营收 +5% 但批价不回升 → 维持 ★★★★
- 营收 <+3% → 降至 ★★★½
- **出厂价首次下调** → 降至 ★★★(25 年首例 = 极强减分)

### 2.2 腾讯 (0700.HK) — 核心仓位

**核心问题**:AI capex 投入产出 + 回购纪律 + 微信生态活力

| 观察点 | 加分 | 减分 |
|--------|------|------|
| **Non-GAAP 净利** | 同比 +10%+ | 同比 <+5% 或负 |
| **游戏收入** | +10%+(王者荣耀 + 三角洲行动) | <+5% |
| **广告收入** | 持续 +20%(视频号放量) | <+15% |
| **AI capex** | 单季 <¥300 亿 | >¥500 亿(失控信号)|
| **回购** | 加速(年化 >¥1,200 亿)| 减速(<¥800 亿)|
| **微信 MAU** | 稳定 13 亿+ | 下降 |

**升级/降级规则**:
- 全部加分 → 维持 ★★★★
- 出现 1 个减分 → 维持 ★★★★
- 出现 2-3 个减分 → 降至 ★★★½

## 3. 中报季工作流(8-9 月 SOP)

### 3.1 中报前(8/1-8/10)

```
Step 1: 提前 1-2 周检查 13+1 投资光谱的中报时间表

```

```
Step 5: 中报披露日 — 立即读取公司公告(港股:HKEX,A 股:巨潮资讯,美股:SEC)

Step 8: 给出"维持/升级/降级"初步判断
```

### 3.3 中报后(9/10-9/30)

```
Step 9: 写 13+1 光谱 v3 中报更新版(类似本框架)
Step 10: 更新每个跟踪提案(智谱/美团/阿里/PDD/茅台)
Step 11: 调整投资组合配置(基于升/降级判断)
Step 12: 更新 proposal-index + 投资光谱 README
```
