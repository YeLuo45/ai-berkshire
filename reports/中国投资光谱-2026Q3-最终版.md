## 5. 三个最有价值的跨公司洞察(终极版)

### 5.1 "估值便宜"不是"投资价值高" — 段永平三持仓拆解

段永平的持仓结构是所有中国投资人的**最佳参照系**:

| 角色 | 标的 | 持仓逻辑 | 仓位含义 |
|------|------|----------|----------|
| **信仰底仓** | 茅台 | 2014 至今 14x 复利,2026-01 加仓 | **10-15% 仓位** |
| **现金牛 + 成长** | 腾讯 | 2026-03 真金白银买 | **核心仓位** |
| **极度低估** | PDD | 2015 天使投资,段永平是黄峥老师 | **5-8% 仓位** |

**翻译**:
- "如果你只能买一个中国股票" = 茅台(确定性)
- "如果可以买两个" = 加腾讯(成长性)
- "如果可以买三个" = 加 PDD(弹性)

→ **段永平的标准下,茅台是"基础中的基础",腾讯是"成长主力",PDD 是"弹性补充"**。

### 5.2 "确定性 + 股息"才是终极安全垫 — 5/1 轮动不成立

5/1 轮动估值分析报告的核心发现:
- **腾讯涨完资金溢出到美团/PDD?不成立**
- **腾讯进入低位(百分位 ≤ 20%)时,才是买入 PDD 的最佳时机**
- **"轮动"逻辑应该是反过来:腾讯触底时,美团/PDD 反弹弹性最大**

→ **不要基于"轮动"决策,要基于"确定性 + 股息"决策**。
- 茅台 4.16% 股息 + 永续年金 = 基础仓位
- 腾讯 PE 12.8x + 3% 股息 + 印钞机 = 核心仓位
- 其他 = 卫星仓位或回避

### 5.3 "现金回报"是芒格最看重的指标

| 公司 | 股东回报 | 评级信号 |
|------|----------|----------|
| **茅台** | **4.16% 股息** | ★★★★ |
| **腾讯** | **3% 股息 + 回购** | ★★★★ |
| **阿里** | ~1% | ★★★ |
| **PDD** | **0%**(减分!) | ★★★½ |
| **美团** | 0%(亏损) | ★★★½ |
| **MiniMax** | 0%(亏损) | 回避 |
| **智谱** | 0%(亏损) | 回避 |
| **寒武纪** | 0%(微利) | 回避 |

→ **段永平的"三持仓"(茅台 + 腾讯 + PDD)覆盖了 4%/3%/0% 三个股息层级 — 这不是巧合,是大师级的配置智慧**
