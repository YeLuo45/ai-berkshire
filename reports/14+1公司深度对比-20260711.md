## 1. 段永平式"四选一"决策树

### 1.1 决策框架(5 问)

```
Q1: 你的核心需求?
   ├─ 稳定收息 → 茅台/招行
   ├─ 收息+成长 → 招行/腾讯
   ├─ 成长为主 → 腾讯
   └─ 弹性最高 → 小米

Q2: 你的年龄/阶段?
   ├─ 20-35 岁(年轻) → 接受波动,小米 + 腾讯
   ├─ 35-55 岁(中年) → 平衡,招行 + 茅台 + 腾讯
   └─ 55+ 岁(退休) → 稳定,茅台 + 招行

Q3: 你能接受 5 年内最大回撤?
   ├─ <10% → 茅台 + 招行(几乎无回撤)
   ├─ 10-20% → 招行 + 茅台 + 腾讯
   ├─ 20-30% → 腾讯 + 茅台 + 小米
   └─ >30% → 小米 + 腾讯(高弹性)

Q4: 你的总投资规模?
   ├─ <50 万 → 极简(2 只标的)
   ├─ 50-200 万 → 标准(5-7 只标的)
   └─ >200 万 → 完整(8-10 只标的)

Q5: 你对未来的判断?
   ├─ 没有 AI 革命 → 茅台(确定性)
   ├─ AI 革命但腾讯是入口 → 腾讯
   └─ AI 革命 + 中国制造业升级 → 小米
```

## 5. 4 个核心标的的"段永平式细节"

### 5.1 茅台 — "永续垄断 + 4.16% 股息"

**一句话**:**"地球上最好的生意之一"** — 92% 毛利率、50% 净利率、30%+ ROE、零负债、4.16% 股息。

**关键事实**:
- 25 年出厂价从未下调
- 2026-03 又提价 8.6%
- 核心产区仅 15 平方公里(地理垄断)
- 5 年生产周期(时间垄断)
- 国酒品牌(心智垄断)

**风险**:出厂价首次下调(25 年首例 = 极强减分)

### 5.2 招行 — "债+看涨期权 + 5.34% 股息"

**一句话**:**"用零售客群换便宜负债,再用客户关系做财富管理"** — PB 0.84x(破净)+ 5.34% 股息(超茅台)+ 财富管理 +21.39%。

**关键事实**:
- 0.84 PB 隐含 9% ROE(已悲观定价)
- 财富管理收入三年来首次反转
- 2.24 亿零售客户 + 17.08 万亿 AUM
- 活期存款占比 49%(行业最低成本负债)

**风险**:净息差日本化(ROE 5-7% → 合理 PB 0.3-0.5x)

### 5.3 腾讯 — "印钞机 + 3% 股东回报"

**一句话**:**"14 亿人离不开的微信"** — PE 12.8x + 3% 股东回报(回购+分红)+ 印钞机。

**关键事实**:
- 微信 14 亿月活(不可复制)
- 游戏 + 广告 + 金融科技三引擎
- 2026-03 段永平真金白银买
- 2025 回购 HKD 800 亿 + 分红 HKD 410 亿

**风险**:AI capex 单季 >¥500 亿(失控信号)

### 5.4 小米 — "4 大飞轮 + SOTP 折价 50%"

**一句话**:**"硬件入口+互联网变现+生态飞轮"** — PE 13-15x(历史最低)+ 净现金 40% + SOTP 估值 45-57 港币。

**关键事实**:
- 4 大业务飞轮同时增长
- 互联网服务毛利 76.5%(真"印钞机")
- 汽车 +224% 增长(2025 交付 41 万辆)
- 净现金 ¥2,326 亿 = 市值 40%

**风险**:2026 是业绩低谷年(手机毛利下滑 + 存储涨价 + 汽车投入期)
