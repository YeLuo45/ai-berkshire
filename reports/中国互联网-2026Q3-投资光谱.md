## 5. 三个最有价值的跨公司洞察

### 5.1 "估值错位"与"确定性强弱"成反比

| 公司 | 估值错位 | 确定性 |
|------|----------|--------|
| 腾讯 | 中度低估 | **极高** |
| 美团 | 重度低估(PS 1x) | 中高 |
| 阿里 | 重度低估(SOTP 51%) | 中低 |
| MiniMax | 大致合理(经历 74% 回撤) | 低 |
| 智谱 | 极度高估(透支 15 年) | 低 |
| 寒武纪 | 极度高估(PE 200+) | 低 |

**翻译**:**估值便宜 ≠ 投资价值高**。需要"估值便宜 + 确定性高"双重满足 — **腾讯和美团同时满足,其他都不满足**。

### 5.2 段永平的明确偏好:腾讯 > 一切

| 信号 | 含义 |
|------|------|
| 腾讯:真金白银买(2026-03 雪球发言) | **唯一明确背书的标的** |
| 茅台 vs 美团:强烈不推荐换仓 | 美团不是替代品 |
| 阿里:"宁愿更高的 PE 买腾讯" | 段永平标准下腾讯胜 |
| MiniMax / 智谱 / 寒武纪:未公开背书 | 这些都是"叙事驱动"标的 |

**翻译**:如果只信段永平,**只买腾讯**。

### 5.3 "现金 + 投资组合"是终极安全垫

| 公司 | 现金 / 投资组合 | 安全垫占市值 |
|------|-----------------|--------------|
| **腾讯** | ¥1,469 亿 + ¥1 万亿组合 | **23%** |
| **美团** | ¥1,170 亿净现金 | ~25% |
| **阿里** | ¥3,169 亿 | ~17% |
| MiniMax | US$16 亿 | ~11% |
| 智谱 | ¥45-55 亿(17 月跑道) | <1% |
| 寒武纪 | 极少 | <1% |

**翻译**:**腾讯和美团有"白送"的安全垫,其他公司没有**。

## 9. 一句话核心结论

> **6+1 投资光谱中,只有腾讯是确定性买入,美团是分批建仓机会,阿里是卫星仓位,其他三家(寒武纪/MiniMax/智谱)都是叙事驱动+估值透支的回避标的。**

> **如果只买一个 → 腾讯**
> **如果愿意承担一定风险 → 腾讯 + 美团**
> **如果是配置型组合 → 腾讯 + 美团 + 阿里**
