## 核心结论

### 生意质量 ★3.2(六小虎中商业化最好,但规模小、模式未验证)

- ✅ 海外收入结构独特(73%),视频模型海螺 7.7%→32.6%
- ✅ 单位经济改善(销售费用降 40% 收入增 159%)
- ⚠️ Talkie/星野 AI 陪伴粘性未经周期验证
- ⚠️ B 端 API 直面 DeepSeek 价格战

### 股票定价 ★1.5/5(176x PS = 任何不涨 7 倍的版本都负 EV)

- 反向估值:维持现价持平(25x PS)需 2028 收入达 US$5.57 亿,CAGR 91.7%
- 赔率结构严重负向 — 这是核心卖出论据

### 综合判断:**回避**(好生意 × 坏价格 = 坏投资)

**真正买入信号**(任一满足才重新评估):
1. 股价跌至 PS<55x(~110 HKD 下方)**且**收入维持 50%+ 增速
2. 8 月中报:B 端占比过半 + 毛利率 30%+ + ARR 客单价稳健
3. 经调整经营现金流转正
4. 解禁后早期股东真实减持公告(暗示聪明钱行为)
