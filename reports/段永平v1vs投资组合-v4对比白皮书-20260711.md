## 1. 段永平"投资组合理论"核心(5 大原则)

### 1.1 5 大原则

| # | 原则 | 核心 |
|---|------|------|
| 1 | **好生意 + 对的人 + 对的价格** | 三者缺一不可 |
| 2 | **"如果你打算持有一只股票 10 年,你不需要每天看它的价格"** | 长期持有 |
| 3 | **"投资是买公司,不是买股票"** | 关注生意本质 |
| 4 | **"好公司尽量不卖"** | 持有好公司 |
| 5 | **"对的事情 + 把事情做对"** | 商业模式 + 执行 |

### 1.2 段永平式"三选一"决策框架

```
Q1: 你的核心需求?
   ├─ 稳定收息 → 茅台/招行
   ├─ 收息+成长 → 招行/腾讯
   └─ 弹性 → 小米

Q2: 段永平背书是否关键?
   ├─ 是 → 茅台 + 腾讯 + PDD
   └─ 否 → 加入 ★★★★ 高确定性标的(招行/小米/京东/网易/宁德)
```

## 3. 4 大理论升级

### 3.1 升级 1:股东回报率是终极安全垫(v2 → v4)

**v1 误区**:
- 段永平背书是必要条件,但不是充分条件
- 段永平没有公开持仓 ≠ 不值得买

**v4 升级**:
- **股东回报率 = 现金分红 / 回购 / 净资产复利**
- 招行 5.34% > 茅台 4.16% > 腾讯 3% > 其他 0%
- **段永平非持仓的 ★★★★ 标的(招行/小米/京东/网易/宁德)有独立价值**

**升级原因**:段永平没买的标的可能是真便宜(京东 10% 股东回报率)

### 3.2 升级 2:股息率是终极现金流(v4)

**v1 → v4 股息率变化**:
- 茅台 4.16%(v1)→ 仍 4.16%(v4)
- 招行 **5.34%(v4 新增)** — 超过茅台
- 腾讯 ~3%(v1)→ 仍 ~3%(v4)
- 小米 0%(v3)→ 仍 0%(v4)
- 美的 **6-7%(v4 新增)** — **最高股东回报**

**关键洞察**:
- 招行 + 美的 = 12% 股东回报(加权)
- 茅台 + 腾讯 = 7% 股东回报
- 段永平 v1 标的 7% 股息,v4 加入招行/美的 升至 9-10%

### 3.3 升级 3:护城河深度加权(v3 → v4)

**v1 标准**:段永平背书(简单,只看 3 家)
**v4 标准**:护城河深度 × 估值 × 股东回报(综合,17 家)

| 护城河类型 | v1 标准 | v4 标准 |
|------------|---------|---------|
| 永续垄断 | ✅(茅台) | ✅(茅台 + 招行)|
| 14 亿人印钞机 | ✅(腾讯) | ✅(腾讯)|
| 客户关系 | ❌(无)| **✅(招行 ★★★★ 新增)** |
| 全球化 | ❌(无)| **✅(美的 + 海尔)** |
| 4 飞轮 | ❌(无)| **✅(小米)** |
| SOTP 折价 | ❌(无)| **✅(京东)** |
| 轻资产 | ❌(无)| **✅(网易)** |
| 行业龙头 | ❌(无)| **✅(宁德)** |

### 3.4 升级 4:跟踪提案体系(v3 → v4)

**v1 → v2**:无跟踪提案
**v3**:5 个跟踪提案(等中报验证)
**v4**:7 个跟踪提案(覆盖 ★★★★ + 关键 ★★★½)

## 5. 段永平式"信仰组合"演化

### 5.1 v1 信仰组合

```
茅台 40% + 腾讯 40% = 防御性 80%
PDD 20% = 进取性 20%
```

**问题**:
- 段永平背书 100% 依赖
- 缺乏"债+看涨期权"型标的
- 缺乏全球化制造业

### 5.2 v4 信仰组合

```
茅台 25% + 招行 15% = 防御性 40%(加入债+期权)
腾讯 25% = 现金牛 25%
小米 10% = 弹性 10%
京东 10% = SOTP 折价 10%
网易 10% = 印钞机 10%
宁德 5% = 等价格 5%
海尔 3-5% = 消费卫星
美的 5-8% = 制造业核心
```

**升级**:
- 加入招行(★★★★ 债+看涨期权)
- 加入小米(★★★★ SOTP 折价 50%)
- 加入京东(★★★★ 10% 股东回报)
- 加入网易(★★★★ 轻资产印钞机)
- 加入宁德(★★★★ 等价格)
- 加入海尔(★★★ 消费卫星)
- 加入美的(★★★★ 制造业核心)

### 5.3 信仰组合演化的核心

| 版本 | 防御 | 卫星 | 进取 | 防御占比 |
|------|------|------|------|----------|
| **v1** | 茅台 40% + 腾讯 40% | PDD 20% | 0% | **80%** |
| **v2** | 茅台 30% + 腾讯 30% | 京东 15% + 网易 15% + 其他 10% | 0% | **60%** |
| **v3** | 茅台 25% + 腾讯 25% | 京东 15% + 网易 10% + 宁德 5% + 其他 10% | 小米 10% | **50%** |
| **v4** | **茅台 25% + 招行 15%** | 京东 10% + 网易 10% + 宁德 5% + 海尔 3-5% + 美的 5-8% | **腾讯 25% + 小米 10%** | **40%** |

**核心趋势**:
- 防御占比从 80% → 40%(逐步多元化)
- 段永平背书占比从 100% → 43%(更独立判断)
- 卫星标的多样化(从 1 个到 6 个)

## 6. 段永平"四不要"原则

### 6.1 不要 1:不要追涨

**v1 → v4 实践**:
- 茅台 PE > 22x 暂停加仓
- 小米 HK$30+ 暂停
- 招行 PE > 18x 暂停

### 6.2 不要 2:不要恐慌

**v1 → v4 实践**:
- 茅台出厂价首次下调=极强减分,不恐慌卖
- 招行 PE < 12x 是黄金买点
- 京东 PE 10x 是机会

### 6.3 不要 3:不要听消息

**v1 → v4 实践**:
- 段永平没有公开持仓 ≠ 不值得买(独立判断)
- "国产替代"叙事(智谱/寒武纪)需验证基本面
- "AI 算力"叙事(中际旭创)需验证 FCF

### 6.4 不要 4:不要频繁交易

**v1 → v4 实践**:
- 每只每周最多 1 次
- 持仓期 3-5 年
- 段永平拷问:如果股市关闭 5 年,愿意持有吗?
